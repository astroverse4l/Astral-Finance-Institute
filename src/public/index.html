AstroVerse API Details

Our API provides the following endpoints:

GET /api/v1/nfts
- Description: Fetch a list of NFTs.
- Request Format: None
- Response: JSON array of NFT data.
- Error Codes: 404 (Not Found), 500 (Server Error)

POST /api/v1/transactions
- Description: Submit a blockchain transaction.
- Request Format: JSON { "from": "address", "to": "address", "amount": "value" }
- Response: Transaction hash.
- Error Codes: 400 (Bad Request), 500 (Server Error)

GET /api/v1/prices
- Description: Fetch real-time crypto prices.
- Request Format: None
- Response: JSON { "symbol": "price" }
- Error Codes: 404 (Not Found), 500 (Server Error)

Our API supports multiple languages, including JavaScript, Python, and Java. Visit our documentation page for code examples and SDKs.
